1. Color

Colores principales

The primary color is the color displayed most frequently across the platform and components.
The secondary color is used to accent select parts of the UI such as buttons, floating acition buttons and progress bars. Each color has a corresponding gradient used for backgrounds.

Base Colors are used for background of the Chat area.

4. Botones

Primary buttons have two functions - as the main call to action for users, and as an action that normally completes the user's task.

A disabled button is used for telling users that an action is available, but not at the moment.

Secondary buttons serve as cues for optional actions that user can take.

Tertiary icons are text links.

Floating Action buttons are associated to users documents, and stick to their position as the user scrolls.

Dev tricks

En esta maqueta, hay algunas clases que se pueden aplicar para ver diferentes versiones de la misma pantalla. A saber:

Home page

  • Aplicar la clase .dev-user-login a la etiqueta <body> para alternar entre las versiones de visitante/usuario registrado del menú. Aclaración: el JS funciona visualmente, pero no deberían coexistir ambos menúes a la vez en la app.

Solicitar limpieza

  • Paso 1: se puede aplicar .step1.localnuevo o .step1.localexistente al elemento div.clean-wizard. Esto controla que se muestren diferentes bloques en el paso, que están englobados por la clase .block-alternative.
